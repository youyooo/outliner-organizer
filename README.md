# Outliner Organizer

A Blender addon that adds one-step collection organization to the Outliner context menu.

Right-click any object in the Outliner and create a collection with the correct name — no more manual "New Collection → Rename → Move" steps.

## Features

- **New Collection & Move** — Create a collection named after the selected object (or the active object for multi-selection) and move everything into it
- **New Collection Each** — Create individual collections for each selected object, each named after its object

## Installation

1. **Download** the addon (clone this repo or download as ZIP)
2. **Blender → Edit → Preferences → Add-ons → Install…**
3. Select `__init__.py` or the ZIP file
4. Enable "Interface: Outliner Organizer"

## Usage

1. Open the **Outliner** (default: top-right panel)
2. **Right-click** one or more objects
3. Go to **Organize** in the context menu
4. Choose:
   - **New Collection & Move** — one collection, named after the right-clicked object
   - **New Collection Each** — one collection per object

## Compatibility

- Blender 4.0+
- Works in View Layer and Scenes display modes
